# 🎚️ **PID 滑杆调整说明（Sliders to Adjust Flight Characteristics）**

这些滑杆用于**调整飞行器的飞行特性（PID 增益）**，从而优化响应性、平稳性与抗扰性能。

| **名称**                                              | **说明**                                                     |
| ----------------------------------------------------- | ------------------------------------------------------------ |
| **Damping（阻尼 / D 增益）**                          | 控制快速运动的抑制程度，减少 P 振荡。较高的 **D 增益** 会让飞行器对摇杆输入反应更钝，电机发热更明显，但能更好地抑制快速振荡、改善桨流（prop wash）表现。较低的 **D 值** 则响应更快，但抗风扰与抗桨流能力会下降。 |
| **Tracking（跟踪性 / P & I 增益）**                   | 提升飞行器的响应性和轨迹保持能力。提高 Tracking 可让飞行器更准确地跟随指令、抵抗外部扰动（如风），避免机头偏离航线。过低的 Tracking 会导致飞行器在桨流中或快速输入时出现“摇晃”和“偏航”；过高的 Tracking 会造成振荡、过度反弹（bounceback）及电机过热。 |
| **Stick Response（摇杆响应 / FF 增益）**              | 控制飞行器对**快速摇杆输入**的反应速度。较低的 Stick Response 会导致响应迟钝，动作滞后，翻滚结束时可能出现慢速反弹；较高的 Stick Response 则反应更灵敏，但可能在翻滚或急停时出现**过冲（overshoot）**。💡 注意：若飞行器权重较大或 Stick Response 设置较低，可通过开启 “I-term Relax” 并适当调整其截止频率（cutoff）来减少反弹。 |
| **Dynamic Damping（动态阻尼 / D Max）**               | 控制 D 项在**快速动作时的最大提升量**。提高 D Max 可让飞行器在快速方向切换时获得更好的超调控制。适用建议：- **竞速机（Race）**：主 Damping 较低以减轻电机热量，可右调此滑杆以提升急转控制力。- **高清或电影机（Cine/HD）**：若前飞时不稳，应调整主 Damping 滑杆，而非 Dynamic Damping。- **花式机（Freestyle）**：尤其是重机型，右调可改善翻滚、空翻的超调控制。⚠️ 注意：如果调整后仍无法改善翻滚超调，可能是由于：• “I-Term Relax” 不足；• 电机失步（desync）；• 推力不足（Motor Saturation）。此时应恢复滑杆至默认位置，并检查根本原因。 |
| **Drift – Wobble（漂移/抖动 / I 增益）**              | 微调 **I 项（积分项）**。提高 I 可改善螺旋飞行、绕圈或低油门（0% throttle）时的航迹保持。但过高的 I（尤其是 P 不足时）可能导致在翻滚结束或拉零油门时出现晃动或反弹。一般应将该滑杆调至尽可能高，使飞行器在螺旋或绕圈时保持轨迹，**但不要高到出现抖动或0油门晃动**。💡 若出现明显反弹，请确认已启用 “I-term Relax”，并尝试降低 `iterm_relax_cutoff` 值。 |
| **Pitch Damping（俯仰阻尼 / Pitch:Roll D 比）**       | 控制俯仰轴相对滚转轴的 D 阻尼增益。适用于在俯仰方向出现过冲或反弹的情况。由于 Pitch 轴惯性更大（机头上下的惯性力矩更高），一般需要更高的阻尼。建议先调好主 Damping 与 Tracking 滑杆（以获得稳定的 Roll 轴表现），再单独微调 Pitch Damping 以优化俯仰控制。 |
| **Pitch Tracking（俯仰跟踪 / Pitch:Roll P, I & FF）** | 控制俯仰轴相对滚转轴的 **P、I、FF** 增益。提高该滑杆可强化俯仰稳定性，减少油门变化或俯仰输入时的机头“点头”晃动。适用于在快速拉俯仰或松油门时出现机头晃动的机型。可同时考虑提高 **Anti-Gravity（抗重力）** 增益。建议先调整主 Damping 与 Tracking，再通过 Pitch Tracking 滑杆细调俯仰响应。 |
| **Master Multiplier（主增益倍数）**                   | 同步增加所有 PID 参数。⚠️ 除非其他滑杆已调到极限，否则**不要轻易调整此滑杆**。通常仅在**低推重比**或**高惯量（MoI）**机型（如 X-Class、CineLifter）上需要。过高的 Master 增益会导致高频振荡（trilling）或电机过热。 |

# ⚙️ **Feed-Forward（前馈控制）**

| **名称**                           | **说明**                                                     |
| ---------------------------------- | ------------------------------------------------------------ |
| **Jitter Reduction（抖动抑制）**   | 当摇杆缓慢移动时，减少 Feedforward 输出，与摇杆位置无关。可在执行平滑圆弧时实现无抖动的流畅飞行；而当摇杆快速移动时，仍可获得全量 Feedforward 响应且无延迟。启用后无需使用 Transition 功能，应将 Transition 设为 0。 |
| **Smoothness（平滑度）**           | Feedforward 平滑滤波用于抑制前馈信号中的噪声。应使用**最小的平滑值**，只要波形足够平滑即可。 |
| **Averaging（平均化）**            | Feedforward 平均化会取最近 2 或 3 个前馈步长的平均值，从而平滑 Feedforward 信号，但会增加轻微延迟。在 RC 链路较快、信号存在上下交替抖动时效果最佳。 |
| **Boost（前馈增压）**              | 在摇杆加速或减速时添加额外推力。减少由电机加速延迟导致的陀螺滞后现象，并在摇杆减速时通过快速降功率抑制过冲。 |
| **Max Rate Limit（最大速率限制）** | 当摇杆接近最大偏移时削减 Feedforward，以减少翻滚或空翻起始阶段的过冲。结束阶段不受影响。数值越低，削减越早开始。 |
| **Transition（过渡）**             | 线性削减摇杆中位附近的 Feedforward。在 Betaflight 4.2 及更早版本中用于柔化中位响应；4.3 起该功能已由 Jitter Reduction 取代，不再推荐使用。数值 0 = 关闭；数值 0.3 = 摇杆居中时前馈为 0，偏移超过 30% 时恢复正常。 |

# 🧩 **其他 PID 相关功能（Others）**

| **名称**                                      | **说明**                                                     |
| --------------------------------------------- | ------------------------------------------------------------ |
| **I Term Relax（I 项放松）**                  | 在快速动作（如翻滚、空翻）时抑制 I 项积累，从而减少结束时的反弹（Bounceback）。**Setpoint 模式** 使用平滑摇杆输入，适合灵敏机型；**Gyro 模式** 适合响应较慢的机型。通常不建议在 **Yaw（偏航）** 上使用。 |
| **Anti Gravity（抗重力）**                    | 在快速油门变化期间（以及短时间之后）提升 I 项（在 4.3 版本中同时提升 P 项），以增强姿态稳定性。较高的增益有助于提升低推重比或重心偏移机型的稳定性。 |
| **I Term Rotation（I 项旋转）**               | 当飞行器持续偏航或执行复杂动作（如滚转、漏斗）时，自动旋转 I 项向量至正确轴向。对目视花式（LOS Acro）飞行员尤其有帮助。 |
| **Dynamic Damping - Gain（动态阻尼增益）**    | 控制飞行器在快速转弯或桨流（Propwash）抖动时，D 项提升的灵敏度。Gain 值越高，D 项越快提升到 D Max。建议值：**40–50**（适合干净的 Freestyle 机型）；较低值仅在极快速动作中提升 D，更适合竞速机。⚠️ 注意：Gain 或 Advance 必须至少设置为 **20 以上**，否则 D 不会提升。两者都为 0 时，D 将锁定在基础值。 |
| **Dynamic Damping - Advance（动态阻尼前馈）** | 增强 Gain 对摇杆快速动作的响应敏感度。它不依赖陀螺或桨流信号，而是**提前作用**。适用于响应滞后的低推力机型。一般建议保持 0。⚠️ 注意：Advance 或 Gain 至少需一个大于 20，否则 D 不会增加。 |

# ⚡ **Throttle 与电机设置（Throttle and Motor Settings）**

| **名称**                                      | **说明**                                                     |
| --------------------------------------------- | ------------------------------------------------------------ |
| **Throttle Boost（油门增压）**                | 在快速油门输入时临时提升油门输出，提高即时响应速度。         |
| **Motor Output Limit（电机输出限制）**        | 限制每个电机的最大驱动信号百分比。用于在高电压电池上保护高 KV 电机与 ESC，减少电流和发热。示例：• 6S 电池搭配原 4S 设备，可尝试 **66%**；• 4S 电池搭配 3S 设备，可尝试 **75%**。⚠️ 确保所有部件能承受所使用电池的电压。 |
| **Dynamic Idle Value [×100 RPM]（动态怠速）** | 改善低转速下的控制精度并降低电机失步风险。提升 PID 控制力、零油门稳定性、倒飞悬停及刹车效果。推荐值：**3000–3500 RPM**。详见 Betaflight Wiki。 |
| **Vbat Sag Compensation（电压下垂补偿）**     | 在电池电压下降时自动提升电机输出，使油门与 PID 响应保持一致。补偿范围 0–100%，建议设置为 **100%（全补偿）**。详见 Wiki。 |
| **Thrust Linearization（推力线性化）**        | 改善低油门时的推力控制与响应性。对 **Whoop** 与 **48kHz ESC** 机型尤其有效。对高油门无影响。推荐补偿：**30–40%**（最大 150%）。 |

# 🧭 **其他杂项设置（Miscellaneous Settings）**

| **名称**                                          | **说明**                                                     |
| ------------------------------------------------- | ------------------------------------------------------------ |
| **Cell Count（电池节数自动切换）**                | 设置电池节数，用于自动切换 PID 配置文件。                    |
| **Acro Trainer Angle Limit（Acro 训练角度限制）** | 为学习 Acro 模式的新手设置最大倾斜角度限制（10°–80°）。需在 Modes 选项卡中通过开关启用。 |
| **Integrated Yaw（积分偏航）**                    | 将偏航轴的 P、I、D 项融合，使其调参方式类似俯仰与横滚。几乎不需要 I 项，因为积分后的 P 行为类似 I，而积分后的 D 行为类似 P。⚠️ 需启用 **Absolute Control** 功能。 |
| **Absolute Control（绝对控制）**                  | 通过在机体坐标系中累积陀螺误差，并混入比例修正项来增强姿态控制。要求启用 **AirMode** 与 **I Term Relax（RP 轴）**。当与 Integrated Yaw 联用时，可启用 **I Term Relax for RPY**。 |
